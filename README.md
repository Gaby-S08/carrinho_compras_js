# Carrinho de Compras

Projeto desenvolvido em 2021, durante aulas de Linguagem e Desenvolvimento Web, do curso Técnico de Informática integrado ao Ensino Médio, no Instituto Federal de Caraguatatuba.

## 🛒 Carrinho de Compras com JavaScript
# 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de um carrinho de compras utilizando HTML, CSS e JavaScript, com foco na prática de manipulação do DOM e lógica de programação.

A proposta foi criar uma aplicação simples onde o usuário pode adicionar produtos ao carrinho, visualizar os itens selecionados e calcular o valor total da compra.

# 🎯 Objetivos
Praticar JavaScript na manipulação de elementos HTML
Trabalhar com eventos (click, input, etc.)
Implementar lógica de adição e remoção de itens
Calcular automaticamente o valor total da compra

# 🧩 Funcionalidades
✔️ Adicionar produtos ao carrinho
✔️ Exibir lista de produtos selecionados
✔️ Atualizar quantidade de itens
✔️ Remover itens do carrinho
✔️ Calcular o valor total automaticamente

# 🛠️ Tecnologias utilizadas
- HTML5
- CSS3 (fornecido no projeto)
- JavaScript
- Git hub
